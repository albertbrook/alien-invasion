### 玩法
左右键移动<br>
空格发射子弹<br>
P键或Play按钮开始游戏<br>
### 说明
每过一关飞船，子弹，外星人都会加速<br>
外星人分值也会提高<br>
### 其它
这是根据《Python编程：从入门到实践》的项目一重新做的<br>
图片都用矩形方块代替了，不需要图片了<br>
整个程序也进行了重构，用面向对象的思想编程<br>
<br>
另外用pyinstaller打包了一个game.exe<br>
不用运行环境也可以直接运行<br>
